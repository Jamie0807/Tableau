# 导出文件

此文件夹用于存放从 Tableau 导出的可视化图表和报告。

## 支持的格式

- **图像**: PNG, JPG
- **文档**: PDF
- **数据**: CSV, Excel
- **交互式**: HTML (Tableau Public)

## 导出指南

### 导出图像
1. 在 Tableau 中打开工作表或仪表板
2. 工作表 → 导出 → 图像
3. 选择分辨率和格式

### 导出 PDF
1. 文件 → 打印为 PDF
2. 选择页面布局和方向

### 导出数据
1. 在工作表中右键点击数据
2. 查看数据 → 导出全部

## 注意事项

- 确保导出的图像分辨率适合最终用途
- PDF 导出会保留交互筛选器的当前状态
- 大型数据导出可能需要较长时间
