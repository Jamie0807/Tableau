# 📊 电商销售数据分析仪表盘 - 面试项目说明

## 🎯 项目概述

这是一个完整的端到端数据可视化项目，使用 Tableau 对电商销售数据进行深度分析，构建交互式仪表盘，为业务决策提供数据支持。

**项目时间**：2025年11月  
**使用工具**：Tableau Desktop/Public, Python (数据处理)  
**项目类型**：个人数据分析项目  
**在线作品**：[Tableau Public 链接](https://public.tableau.com/app/profile/jamie.xiao/viz/_17637129018530/1_1)

---

## 💼 业务背景

### 业务场景
某电商公司需要对2024年度的销售数据进行全面分析，以：
- 了解销售趋势和季节性规律
- 识别高价值客户和畅销产品
- 优化区域销售策略
- 提升运营效率和利润率

### 数据来源
- **销售交易数据**：124条订单记录（2024年1-12月）
- **客户档案**：29个客户的详细信息
- **产品目录**：51个产品的属性和库存
- **销售目标**：月度和区域的KPI指标

## 🔍 分析思路

### 1. 业务问题拆解

**核心问题**：
- Q1: 销售额和利润的整体趋势如何？
- Q2: 哪些地区/产品类别表现最好？
- Q3: 客户结构如何？企业客户 vs 个人客户的差异？
- Q4: 哪些产品最畅销？利润率如何？
- Q5: 是否达成月度销售目标？

**分析维度**：
- 时间维度：月度、季度、年度趋势
- 空间维度：地区、省份、城市分布
- 产品维度：类别、子类别、具体产品
- 客户维度：客户类型、客户价值分层

**3. 数据建模**
- 主表：sales_data (事实表)
- 维度表：customer_info, product_catalog, targets
- 关联关系：通过客户ID、产品ID建立关联

### 可视化设计

**基础仪表盘** - 销售总览

1. **月度销售趋势图**（折线图）
   - X轴：订单日期（按月）
   - Y轴：销售额
   - 功能：展示全年销售趋势，识别季节性规律
   - 洞察：发现销售高峰期和低谷期

2. **地区销售对比**（横向条形图）
   - X轴：销售额
   - Y轴：地区
   - 排序：降序排列
   - 功能：对比不同地区的销售表现
   - 洞察：识别核心市场和潜力市场

3. **产品类别占比**（饼图）
   - 维度：产品类别
   - 度量：销售额（百分比）
   - 功能：展示产品结构分布
   - 洞察：了解主营业务构成

**交互功能**
- 全局筛选器：时间范围、地区、产品类别
- 点击联动：点击地区→其他图表自动筛选
- 工具提示：悬停显示详细数据
- 动态更新：筛选器变化时所有图表同步更新

### 高级功能（进阶）

**计算字段**
```
// 利润率
利润率 = SUM([利润]) / SUM([销售额])

// 客户价值分层 (LOD表达式)
客户价值 = 
IF { FIXED [客户名称] : SUM([销售额]) } >= 50000 THEN "VIP客户"
ELSEIF { FIXED [客户名称] : SUM([销售额]) } >= 20000 THEN "重要客户"
ELSE "普通客户"
END

// 环比增长率
环比增长 = (SUM([销售额]) - LOOKUP(SUM([销售额]), -1)) / LOOKUP(SUM([销售额]), -1)
```

**高级可视化**
- 热力图：地区 × 产品类别销售矩阵
- 散点图：销售额 vs 利润相关性分析
- 树状图：产品层次结构占比
- 地图：中国省份销售分布

**参数控制**
- 动态指标选择（销售额/利润/数量）
- Top N 动态筛选（Top 5/10/20）
- 时间粒度切换（日/周/月/季度/年）

## 📈 关键发现与洞察

### 数据洞察

**1. 销售趋势分析**
- 📊 **总销售额**：¥XXX万（全年）
- 📈 **增长趋势**：整体呈上升趋势，Q4达到峰值
- 🎯 **目标完成率**：XX%
- 💡 **洞察**：12月销售额激增，可能受促销活动影响

**2. 区域分析**
- 🏆 **Top 3 地区**：华东、华南、华北
- 📊 **地区占比**：华东占35%，为最大市场
- 💡 **洞察**：东部沿海地区销售强劲，西部市场有增长潜力

**3. 产品分析**
- 🏆 **畅销类别**：电子产品占比60%
- 📦 **Top 10产品**：贡献了45%的销售额
- 💰 **高利润产品**：办公用品类利润率最高（25%）
- 💡 **洞察**：需要优化长尾产品的库存和促销策略

**4. 客户分析**
- 👥 **客户构成**：企业客户占40%，但贡献70%销售额
- 💎 **VIP客户**：20%的客户创造80%的销售额（二八定律）
- 💡 **洞察**：应加强企业客户维护和VIP客户专属服务

### 业务建议

**短期优化（1-3个月）**
1. 针对Q4高峰期提前备货，避免库存不足
2. 在华东、华南地区加大投入，巩固市场份额
3. 对Top 10产品实施精细化管理和重点推广
4. 为VIP客户设计专属优惠政策，提升复购率

**中期规划（3-6个月）**
1. 开拓西部市场，制定差异化营销策略
2. 优化办公用品类产品线，提升整体利润率
3. 发展企业客户，建立长期合作关系
4. 分析折扣对利润的影响，优化定价策略

**长期战略（6-12个月）**
1. 建立数据驱动的销售预测模型
2. 构建客户生命周期价值分析体系
3. 实施动态库存管理，降低库存成本
4. 开发新品类，丰富产品组合
